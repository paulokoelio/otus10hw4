Detailed Description

Заголовочный файл с определениями шаблонных функций для tuple.

Заголовочный файл с определением функций print_ip() для данных типа tuple, а также с определением универсальной вспомогательной функции for_each(tuple , callback) производящей обход всех элементов tuple во время компиляции и обрабатывающей их функцией callback. Код функций для tuple основан на исходном коде представленном на сайте: http://artlang.net/post/c++11-obkhod-elementov-kortezhe-std-tuple/
